# Project Structure

A detailed explanation of every file and directory in the project.

---

## Root Directory

| File | Purpose |
|---|---|
| `app.py` | **Main application.** Streamlit web app that loads the trained model and provides a drag-and-drop interface for X-ray classification |
| `colab_data_setup.py` | **Data pipeline script.** Designed for Google Colab — downloads the Kaggle dataset, filters hidden files, and splits data by patient ID to prevent leakage |
| `requirements.txt` | Python package dependencies |
| `run_app.bat` | **Windows launcher.** Double-click to activate the virtual environment and start the app |
| `run_app.sh` | **Bash launcher.** Same as above, for Git Bash or Linux/Mac |
| `README.md` | Project overview and setup guide |
| `LICENSE` | MIT License |
| `.gitignore` | Files excluded from version control |

---

## `/src/`

| File | Purpose |
|---|---|
| `optimize_model.py` | **(Optional)** Converts the PyTorch `.pth` model to ONNX format and applies INT8 quantization for faster CPU inference |

---

## `/notebooks/`

| File | Purpose |
|---|---|
| `Colab_Model_training.ipynb` | **Training notebook.** Contains the complete pipeline: environment setup, data loading, DenseNet121 fine-tuning, evaluation (confusion matrix, classification report), and Grad-CAM visualization. Meant to run on Google Colab with a T4 GPU |

---

## `/docs/`

| File | Purpose |
|---|---|
| `MODEL_CARD.md` | ML Model Card with architecture, performance metrics, limitations, and ethical considerations |
| `PROJECT_STRUCTURE.md` | This file |

---

## `/models/`

| File | Purpose |
|---|---|
| `model_fp32.onnx` | *(Generated)* Full-precision ONNX model |
| `quantized_model.onnx` | *(Generated)* INT8 quantized ONNX model for fast CPU inference |

> These files are generated by `src/optimize_model.py` and excluded from version control.

---

## `/data/`

Empty directory placeholder. The actual dataset is downloaded at runtime via `colab_data_setup.py` on Google Colab.

---

## `/test_images/`

Sample X-ray images for quick testing of the deployed app.

---

## `/.streamlit/`

| File | Purpose |
